# 📖 About

This repository contains the complete implementation of the **Machine Learning Laboratory (PCCSL508)** for **KTU B.Tech Computer Science & Engineering Semester 5**.

The experiments are implemented using **Python**, **NumPy**, **Pandas**, **Matplotlib**, and **Scikit-Learn**, following the latest KTU syllabus.

The objective is not just to complete the lab, but to understand how machine learning algorithms work using real-world datasets, visualization, and performance evaluation.

# 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/ML-LAB.git
```

Move into the folder

```bash
cd ML-LAB
```

Install required packages

```bash
pip install numpy pandas matplotlib scikit-learn notebook
```

Run Jupyter Notebook

```bash
jupyter notebook
```
